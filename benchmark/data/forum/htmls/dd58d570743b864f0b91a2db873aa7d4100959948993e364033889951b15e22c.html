Hi @klwild,

kannst du bitte noch einmal genau beschreiben, was du anpassen möchtest? Die Schriftgröße sollte eigentlich von deinem Theme übernommen werden. Hast du die Sidebar in deinem Theme aktiviert?

Ohne die Seite zu sehen ist es etwas schwierig dir ein passendes CSS zu geben. Um alle Titel der Widgets anzupassen kannst du folgendes CSS verwenden:

h2.widget-title {
font-size: 15px;
}

Je nach Aufbau deines Themes musst du ggf. auch noch !important verwenden:

h2.widget-title {
font-size: 15px !important;
}
