前言#

每次打开别人新的项目都要 npm 依赖都会各种报错很烦

解决#

安装 depcheck

npm i -g depcheck

检查依赖

depcheck

找到 Missing dependencies 缺失依赖

安装缺失依赖
npm i aaa.js bbb.js
